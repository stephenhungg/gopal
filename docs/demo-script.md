# demo script

## setup

1. Start the server with `npm run dev`.
2. Open `http://localhost:3000`.
3. Click `wake gopal`.
4. Allow camera and microphone.

## 5 minute flow

1. Gopal wakes up and greets the user.
2. User asks: "what do you see?"
3. Gopal reacts to the room from the camera frame.
4. User points camera at a sign, object, hallway, or desk clutter.
5. Gopal calls out something useful or funny.
6. User asks for guidance: "where should I go?" or "what should I watch out for?"
7. Gopal gives short spatial guidance in goblin style.

## guardrails

Gopal can be feral and funny, but the demo should avoid identifying real people, sexualizing strangers, or insulting vulnerable people. Keep the chaos aimed at objects, hazards, weird vibes, and the goblin's own theatrics.
